# Crypto-Key-Generator
This tool can be used to create key which can be further be used to encrypt any data.
  
This can be used to make passwords too.
 
# Installation
Unzip this file.
  
Oen a terminal in the same directory as rhe files.
  
Run the command :
  
chmod +x rand.sh
  
then :
  
bash rand.sh
  
or
  
./rand.sh
